# Invoice App Website

A professional, modern landing page for the Invoice App with a clean design and smooth animations.

## Features

- **Responsive Design**: Works perfectly on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional design with smooth animations
- **Feature Showcase**: Highlights all app capabilities
- **Direct Download**: One-click download button for the Android APK
- **Smooth Animations**: Interactive elements with smooth transitions
- **SEO Optimized**: Meta tags and semantic HTML for better search visibility

## Structure

```
InvoiceAppWebPage/
├── index.html          # Main HTML file
├── styles.css          # All styles and responsive design
├── script.js           # Interactive features and animations
├── app-release.apk     # Your Android app file
└── README.md           # This file
```

## How to Use

1. **Local Testing**: Simply open `index.html` in your web browser
2. **Deploy**: Upload all files to your web hosting service

## Deployment Options

### Option 1: GitHub Pages (Free)

1. Create a GitHub repository
2. Upload all files
3. Go to Settings → Pages
4. Select main branch and save
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Free)

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the folder
3. Your site will be live instantly

### Option 3: Vercel (Free)

1. Go to [vercel.com](https://vercel.com)
2. Import your project
3. Deploy with one click

### Option 4: Traditional Web Hosting

Upload all files to your web host via FTP/cPanel

## Customization

### Update Content

- Edit text in `index.html`
- Modify colors in `styles.css` (see CSS variables at the top)
- Add your logo by replacing the SVG in the navbar

### Change Colors

Edit the CSS variables in `styles.css`:

```css
:root {
  --primary-color: #6366f1; /* Main brand color */
  --primary-dark: #4f46e5; /* Darker shade */
  --secondary-color: #f59e0b; /* Accent color */
}
```

### Add Analytics

Add Google Analytics or other tracking code before the closing `</head>` tag in `index.html`
